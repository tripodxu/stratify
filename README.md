# Stratify

**基于多智能体的阶层社会随机模拟系统。**

通过随机生成个体属性（天赋、努力），设定阶层系数与竞争规则，运行大规模模拟，观察社会结构是否能自然涌现出纺锤型、金字塔型、沙漏型等典型形态。

---

## 核心问题

> 仅凭天赋、努力、出身（阶层系数）三要素，加上竞争与寿命机制，能否在没有中央调控的条件下，自发演化出与现实社会相似的结构？

---

## 系统概览

### 个体属性

| 属性 | 符号 | 说明 |
|------|------|------|
| 阶级 | class | 所处的社会阶层 |
| 财富值 | Value | 核心资源，初始为 0 |
| 天赋值 | ge | 先天能力，正态分布随机生成 |
| 努力值 | hard | 后天勤奋程度，正态分布随机生成 |
| 寿命 | life | 正态分布随机生成，受阶层影响实际消耗 |

### 核心公式

`
每 tick 积累速度 = env(阶层系数) × (ge + hard) / 2
`

- **env**：阶层越高，系数越大 → 积累越快
- **ge（天赋）**：决定上限
- **hard（努力）**：决定趋近上限的速度，有隐性寿命代价

### 四大机制模块

`
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  积累与维持  │    │  跨越与恢复  │    │  竞争与掠夺  │    │  寿命与生死  │
│             │    │             │    │             │    │             │
│ env×(ge+hard)│    │ cross 值    │    │ 同层: 公平   │    │ 正态初始寿命 │
│ 需求线检查   │    │ 重返减免    │    │ 跨层: 衰减   │    │ 阶层 decay   │
│ 被动跌落    │    │ 减免衰减    │    │ 高打低: 削减  │    │ 努力过劳代价  │
│ 主动降阶    │    │             │    │ 低打高: 加成  │    │ 代际更替    │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
`

## 目标社会结构

| 结构 | 特征 | 关键驱动 |
|------|------|---------|
| 纺锤型 | 中间阶层庞大 | 适中 cross + 中等流动性 |
| 金字塔型 | 底层庞大，逐层收窄 | 高 cross + 低流动性 |
| 沙漏型 | 中间层塌陷，两极分化 | 高竞争频率 + 中产高 decay |
| 橄榄型 | 中产为主体，稳定繁荣 | 低 cross + 低传承 + 适中 decay |
| 种姓/封建 | 阶层固化，几乎无流动 | 极高 cross + 高传承 |
| 赛博朋克 | 顶层垄断，底层无通道 | 极端 env 差距 + 高向下掠夺 |

## 快速开始

`ash
# 克隆仓库
git clone git@github.com:tripodxu/stratify.git
cd stratify

# 安装
pip install -e .

# 运行测试
python -m pytest tests/ -v

# 运行阶段一实验
python -m stratify.main --ticks 1000
`
