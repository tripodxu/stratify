# Stratify

**一个基于多智能体的阶层社会模拟系统。**

通过随机生成个体属性（天赋、努力），设定阶层系数与竞争规则，运行大规模模拟，观察社会结构是否能自然涌现出纺锤型、金字塔型、沙漏型等典型形态。

---

## 核心问题

> 仅凭天赋、努力、出身（阶层系数）三要素，加上竞争与寿命机制，能否在没有中央调控的条件下，自发演化出与现实社会相似的结构？

---

## 系统概览

### 个体属性

每个个体拥有以下属性：

| 属性 | 符号 | 说明 |
|------|------|------|
| 阶级 | class | 所处的社会阶层 |
| 财富值 | Value | 核心资源，初始为 0 |
| 天赋值 | ge | 先天能力，正态分布随机生成 |
| 努力值 | hard | 后天勤奋程度，正态分布随机生成 |
| 寿命 | life | 正态分布随机生成，受阶层影响实际消耗 |

### 核心机制

`
积累速度 = env(阶层系数) × f(ge, hard)
`

- **阶层系数（env）**：阶层越高，积累越快
- **天赋（ge）**：决定上限
- **努力（hard）**：决定趋近上限的速度，但有隐性寿命代价

### 四大机制模块

**1. 积累与维持**
- 每个时间单位，个体按积累速度增加 Value
- 每个阶层有 Value 需求线，低于需求线持续一定时长将被动跌落
- 新晋者享有时间缓冲期

**2. 跨越与恢复**
- 向上跨层需消耗 cross 值
- 曾经的高阶层者享有重返减免（阶梯型衰减）

**3. 竞争与掠夺**
- 同层竞争：无额外系数，公平对抗
- 跨层竞争：发起方受系数衰减，被竞争方获得加成
  - 高打低：胜后收获也被削减
  - 低打高：胜后收获获得加成
- 资源转移存在系统损耗（败者失去 > 胜者获得）

**4. 寿命与生死**
- 初始寿命正态分布
- 阶层越低，寿命损耗越快
- 高努力值带来额外寿命损耗
- 个体死亡后由新生儿补充，支持代际传承

## 目标社会结构

通过参数调节，尝试复刻以下六种典型社会形态：

| 结构 | 特征 | 关键驱动 |
|------|------|---------|
| 纺锤型 | 中间阶层庞大 | 适中的 cross 值 + 中等流动性 |
| 金字塔型 | 底层庞大，逐层收窄 | 高 cross 值 + 低流动性 |
| 沙漏型 | 中间层塌陷，两极分化 | 高竞争频率 + 中产高 decay |
| 橄榄型 | 中产为主体，稳定繁荣 | 低 cross 值 + 低传承 + 适中 decay |
| 种姓/封建 | 阶层固化，几乎无流动 | 极高 cross 值 + 高传承 |
| 赛博朋克 | 顶层垄断，底层无通道 | 极端 env 差距 + 高向下掠夺 |

## 快速开始

### 环境要求

- Python 3.10+
- NumPy
- Matplotlib / Seaborn（可视化）
- Pandas（数据分析）

### 运行

`ash
# 克隆仓库
git clone git@github.com:tripodxu/stratify.git
cd stratify

# 安装依赖
pip install -r requirements.txt

# 运行基础模拟
python main.py
`
